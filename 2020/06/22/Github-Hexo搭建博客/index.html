Github + Hexo搭建博客

Posted on 2020-06-22

一、相关软件安装与配置

  1. Git

    1. 从Git官网下载软件并安装

    2. 右键在Git Bash命令行中设置用户名与邮箱地址

    3. 创建密钥与配置

      执行ssh-keygen -t rsa -C "<yourEmail@example.com>"

      在C:\Users\<用户名>\.ssh\id_rsa.pub中生成密钥文件

    4. 配置远程密钥

      Github个人中心——>Settings——>SSH and GPG keys——>New SSH keys：填入生成密钥

    5. 创建Github仓库

      新建一个名为userName.github.io的仓库，userName之后会成为你博客地址的一部分

      在仓库settings页面中的GitHub Pages栏中绿色区域就为博客的网址

  2. Hexo

    1. 从Hexo官网下载软件并安装

    2. 在cmd命令行运行npm install -g hexo进行安装

    3. 初始化Hexo：新建你的本地博客文件夹，在文件夹中使用cmd运行hexo init命令初始化博客，会得到如下文件
